# Foodcard

A responsive recipe card app built with React, Vite, and styled-components.

## Features

- Live recipe search from TheMealDB
- Search by ingredient or cuisine
- Random recipe discovery
- Diet preferences for vegan, vegetarian, pescatarian, or everything
- Supabase-backed cookbook storage
- Responsive mobile and desktop layout

## Local setup

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and add Supabase credentials to enable cookbook saving.
